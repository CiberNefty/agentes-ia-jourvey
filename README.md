# 🤖 Agentes IA Journey

Mi camino aprendiendo a construir agentes de IA con Python y Ollama.
Proyecto en progreso — un commit a la vez.

## 🛠️ Stack
- Python 3
- Ollama (IA local, sin internet, sin costo)
- Llama3.2 (modelo de lenguaje local)
- Google Antigravity (IDE con agentes)

## 📁 Proyectos

### 01 - Hola Ollama
Primer script que se comunica con una IA local.
- Envía mensajes a Ollama via API REST
- Recibe y muestra respuestas en consola
- 100% local, sin internet

## 🚀 Cómo correrlo Por Terminal (GitBash-PowerShell)

1. Instalar Ollama desde ollama.com (winget install Ollama.Ollama) 
2. Correr el modelo: `ollama run llama3.2`
3. Instalar dependencias: `pip install requests`
4. Ejecutar: `python 01_hola_ollama.py`
